How It Works

  1. Encode: Your image is resized, quantized, and converted to a binary stream.
  2. Self-Describe: A header with image metadata is embedded in the code itself.
  3. Render: The binary stream becomes a visible pixel grid (SD-IDC code).
  4. Decode: Any SD-IDC decoder reads the header to reconstruct the original image.
